# Ritika Biswas - Software Engineer Portfolio

Welcome to the repository for my personal developer portfolio! This is a modern, responsive, and interactive single-page application built to showcase my skills, projects, and academic journey.

## 🚀 Live Demo
*(You can add your GitHub Pages link here once deployed, e.g., https://ritika2003biswas.github.io/portfolio)*

## ✨ Key Features
- **Oceanic UI Design**: A custom, multi-layered animated wave background with deep sea blue and purple gradients.
- **AI Chat Assistant**: An integrated, custom-built AI Agent programmed to answer questions specifically about my professional background (Skills, Education, Projects).
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile views.
- **Glassmorphism Components**: Modern blurred, semi-transparent overlays for an elegant feel.
- **Direct Email Integration**: Contact and Resume Request forms designed to bypass local security blocks by directly composing a message in the user's Gmail client.
- **Developer Aesthetic**: Features a custom typewriter-effect logo and clean typography (Inter & Outfit fonts).

## 🛠️ Technology Stack
- **HTML5**: Semantic and accessible structure.
- **CSS3 (Vanilla)**: Advanced styling, CSS variables, keyframe animations, and Flexbox/CSS Grid layouts. 
- **JavaScript (Vanilla)**: DOM manipulation, scroll reveal animations, and AI logic simulation without external API dependencies.
- **Lucide Icons**: High-quality SVG icons.

## 📁 Project Structure
- `index.html`: The main entry point containing all sections.
- `css/style.css`: The complete design system and responsive styles.
- `js/script.js`: Interactive logic (AI Assistant, scroll animations, mobile menu, and form handling).
- `img/`: Folder containing project placeholder images.

## 💡 Running Locally
No server or build process is required! 
Simply double-click `index.html` or open it in any modern web browser to view the portfolio.
